Tracker

The downwind foiling tracker app for you and your crew.

Features:
  • GPS tracking for groups
    • As many crew member and watchers as you want
    • See every crew member in a list with direction and distance or on a map
  • GPS tracking for yourself
    • With statistics
Planned features:
  • Apple Watch support
  • Universal web app
  • Set waypoints for runs
  • Set start and finish for runs

This app was developed with downwind foiling in mind, but you can track any outdoor activity.

If you want to help testing it, let us know (iPhone only at the moment)!
